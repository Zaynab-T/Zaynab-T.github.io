# Personal Portfolio Website
This is a simple personal website created using GitHub Pages to showcase my background, skills, and projects as a Computer Science student.

## Overview
This website was developed as part of an extra credit assingment for my Web Programming class and serves as a starting point for my personal portfolio. It includes basic facts about me, as well as links to my GitHub, resume, and contact information.

## Features
- Introduction/about me section
- Basic styling using CSS
- Links to GitHub, LinkedIn, and email
- Hosted using GitHub Pages

## Technologies Used
- HTML
- CSS
- GitHub Pages
